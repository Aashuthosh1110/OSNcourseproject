# Logs Directory

This directory contains log files generated by the Docs++ system components.

## Log Files

- `name_server_<port>.log` - Name Server operational logs
- `storage_server_<port>.log` - Storage Server operational logs  
- `client_<username>.log` - Client session logs

## Log Format

Each log entry includes:
- Timestamp
- Component name
- Log level (DEBUG, INFO, WARNING, ERROR, CRITICAL)
- Message with relevant context

## Log Retention

Log files are rotated automatically when they exceed size limits. Configure log retention policies in the logging configuration.
